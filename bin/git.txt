git init : được sử dưng để tạo, khởi tạo một kho chứa Git mới (Git Repo) ở local - dùng cho lần đầu
git config --global user.name "ten" : tạo tên lưu lại khi chỉnh sửa
git config --global user.email tenemail : lưu email tk github để làm việc
git status : để biết thông tin trạng thái sửa đổi, thêm mới, xóa các file trước khi thực hiện commit
git add file1 file2 ... : thêm file
git commit -m "ghi chú": thêm ghi chú 
git push -u : đẩy lên github lần đầu            
         -all : đẩy lên tất cả các branch 
git checkout "ten_nhanh" : đổi nhánh
git pull : cập nhật dữ liệu


Note : thêm origin "tên branch" : thêm vào nhánh cụ thể
